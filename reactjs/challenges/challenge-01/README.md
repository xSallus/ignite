# **Task List**
Manage your tasks, anytime, anywhere.

## Setup and run
Clone this repo, then install dependencies by running:
```bash
$ yarn
```
After setup is complete, you're ready to go, just run:
```bash
$ yarn dev
```

## Features

[✓] Add new task

[✓] Mark a task as done

[✓] Remove a task from the list
